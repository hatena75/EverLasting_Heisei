●このゲームについて
グ○ディ○ス×ジ○ウ的ネタシューティングゲーム
平成最後の冬、30年続いたその力で、新たな時代をぶっ飛ばせ！


●基本的な遊び方
敵を倒しつつ、少しでも平成時代を長くしよう！
きつくなってきたら、クルミを取って左クリックで自機を強化！
もう無理だと思ったら、Rキーを押してそれまでのスコアを獲得しよう！
やられてしまうとスコアは0になってしまうので注意！


●ゲームシステム詳細

○平成カウント
生き残っている間、一定時間で左上にある平成の年度が進みます。
年度が進むにつれて強力な敵が出現したり、敵の数が増えたりします。

○生＃退位システム
Rキーを押してリタイア宣言を行うと、2年後にゲーム終了となります。
これによりスコア(年度)が確定し、年度によってコメントや称号が表示されます。
コメントや称号は34種類あります。

○アイテム
黄色の敵を倒すとクルミが出現します。
クルミを取ると下のアイテム枠が1つ右に進みます。
左クリックでアイテムを使用、アイテム枠の進みは初期化されます。

■アイテム詳細
昭和壱:東京タワー(貫通弾)、一定時間で効果が切れる。使用時は通常弾(デフレスパイラル)は出なくなる。

昭和弐:バブル(拡散弾)、崩壊時に弾が4方向に発射される。バブルは敵に当たらない。

大正壱:鷲のマーク(スピードアップ)、自機の移動スピードと年が経つのが速くなる。3回まで使える。

大正弐:選挙投票用紙(サブ弾)、斜めに弾を発射する。2回まで使える。

明治壱:文明開化(ボム)、ボムの所持数が1増える。右クリックで使用できる。最大3個まで所持できる。

明治弐:チョコレート(オプション)、自機と全く同じ動き、攻撃を行うオプションが追加される。

○敵
敵は一発で倒せる敵と、そうでない敵がいます。
複数回攻撃しないと倒せない敵は、別の敵を攻撃として出してきます。
